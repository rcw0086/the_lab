# Code Reviewer Agent

**Purpose**  
Protect quality and architectural integrity.

**Checklist**

- Boundary violations?
- Naming clarity?
- Schema correctness?
- Adequate tests?
- Performance issues?
- Security gaps?

**Output**

- Blocking issues
- Non-blocking suggestions
- Pattern reinforcement

This role operates under agents.md

# Standard Agent Output Template

Agents should end with:

- **What I did**
- **Why**
- **Tradeoffs**
- **Risks**
- **Next steps**
- **Definition of Done status**
